How to Reproduce Everything from Scratch

Prerequisites

  • MATLAB R2025b + Simulink (for data collection)
  • Python 3.x + PyTorch, scikit-learn, numpy, pandas, matplotlib, reportlab

Step 1 — Collect Data from Simulink

matlab -r "run scripts/collect_training_data.m; run scripts/collect_validation_data.m"

Step 2 — Train Phase 1 Baseline

python3 scripts/train_rom.py
python3 scripts/validate_rom.py
python3 scripts/generate_c_code.py

Step 3 — Train Phase 2 Compression Suite (7 models)

python3 scripts/train_all_models.py

Step 4 — Train Phase 3 Advanced Compression

python3 scripts/train_phase3.py

Step 5 — Validate C Code Against Simulink

matlab -r "run scripts/simulink_vs_c_rom.m"
python3 scripts/validate_c_code.py

Step 6 — Generate Reports

python3 scripts/generate_v3_report.py
python3 scripts/generate_comparison_report.py

Using the Recommended C Model in Your ECU

Copy src/rom_lstm_qat.h and src/rom_lstm_qat.c into your embedded project. No other dependencies required.
#include "rom_lstm_qat.h"

// Allocate state once (static or on heap)
ROM_lstm_qat_State state;

// Initialise at startup
ROM_lstm_qat_Reset(&state);

// Call once per control cycle (e.g. 10 ms)
float torque_Nm = ROM_lstm_qat_Step(
    &state,
    air_charge_g_per_s,    // e.g. 0.5 – 3.0 g/s
    speed_rad_per_s,       // e.g. 100 – 600 rad/s
    spark_advance_deg      // e.g. 5 – 35 deg
);

Important Engineering Notes

  • NARX-Ridge closed-loop: When used in closed-loop, NARX-Ridge can drift. Always reset state if the input trajectory deviates significantly from training conditions.
  • Delta poly normalisation: The delta LSTM-8 residual is trained on normalised deltas. At inference, sum ROM_delta_poly_Predict() + ROM_lstm_delta8_Step() before denormalising.
  • Q16 path for no-FPU MCUs: Use rom_lstm_16_q16.{h,c} for MCUs without FPU. All arithmetic uses int32_t accumulators and int16_t weights.
  • MATLAB R2025b API: The From Workspace block uses 'Interpolate','off' not the removed 'ZeroOrderHold','on'. Signal logging must be set on the PORT handle, not the line handle.
